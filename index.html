How to Use ZipTree
Getting Started

You can upload files in two ways:

  • ZIP Files:
    • Click "Choose ZIP File" button
    • Drag and drop a ZIP file into the upload area
  • Project Folders:
    • Click "Choose Folder" button to select a project folder
    • Browse and select any folder from your computer
Features
  • File Preview: Click on any file to view its contents
  • Search: Use the search bar to find files and folders
  • Navigation: Use breadcrumbs to navigate the file structure
  • Download: Click the download icon to save individual files
  • Copy Structure: Copy the entire file structure to clipboard
  • Theme: Toggle between light and dark mode
Tip: Use the expand/collapse buttons to manage the file tree view
For security reasons, executable files (.exe, .bat, .cmd) are not allowed.
